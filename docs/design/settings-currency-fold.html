Currency fold: header shows the active code as the subtitle. Tapping opens the card, clears any prior search, reveals a search field (filters by code substring, case-insensitive) then the full pill grid. Picking a currency calls setCurrency() and folds the card back automatically. The full list now covers ~45 ISO 4217 codes (Asia-Pacific, Americas, Europe, Middle-East & Africa).
Implementation note: SUPPORTED_CURRENCIES expanded in src/features/settings/repository.ts; the search filter is a useMemo in the settings screen — no domain logic change, just a UI fold + filter over the same constant.
