ProjectXavier — workflow mockups rev 8

User flows through the app as currently implemented on main. Each card is a screen; arrows are user actions / navigation. Routes use Expo Router file paths.

rev 2: accounts de-typed (net worth = signed sum of all balances; optional cosmetic tags) · transfer type. rev 3: transfer amounts grey · currency is a single app-level setting (no FX) · optional note per transaction. rev 4: payee & category are searchable comboboxes with inline create · picking a payee auto-fills its first-used category · assistant reconciles via fuzzy-merge + learned default. rev 5: dashboard is a period overview — period button opens a Period dialog (Month/Year/Date); shows net worth at period end + per-account trend chart, income/expense/net, and each account's closing balance rolled forward from the prior period. rev 6: clean screens — add/edit moved into bottom-sheet dialogs; Transactions get a floating + (bottom-right) and tap-to-reveal search; Manage accounts gets + then 🔍 top-right; tap a row to edit (archive from the sheet). rev 7: consistent period scoping — Transactions tab gets the same period button/dialog and shows only the period's entries; account detail opened from the dashboard is period-scoped (balance as of period end + that period's transactions), while Manage accounts → View transactions stays all-time. Manage-accounts management itself is not period-scoped. rev 8: animated avatar — Xavier is an SVG "pet" (Reanimated) that's always subtly alive (breathe + blink) and reacts to assistant state: idle / listening / thinking / happy / confused. (Phase 1; Lottie/Rive is a later option.)

3 · Period overview (Dashboard)

The dashboard is an overview of the selected period. A period button (top-left) opens the Period dialog (Month / Year / Date). For the period it shows net worth at the period end with a per-account trend chart, then income / expense / net savings, then each account's closing balance. Balances roll forward: a period's start = the previous period's closing balance, plus that period's transactions.

4 · Drill into an account

Accounts are listed together — no asset/liability split. Net worth is the signed sum of every balance. Tap an account to drill in. Entry point sets the scope: from the Dashboard the detail is period-scoped (balance as of the period end + only that period's transactions); from Settings → Manage accounts → View transactions it's all-time (current balance + every transaction).
