ProjectXavier — workflow mockups rev 4

User flows through the app as currently implemented on main. Each card is a screen; arrows are user actions / navigation. Routes use Expo Router file paths.

rev 2: accounts de-typed (net worth = signed sum of all balances; optional cosmetic tags) · transfer type. rev 3: transfer amounts grey · currency is a single app-level setting (no FX) · optional note per transaction. rev 4: payee & category are searchable comboboxes with inline create · picking a payee auto-fills its first-used category · assistant reconciles via fuzzy-merge + learned default (new payees created silently on Save).

📱 Tab screen 🗂️ Stack screen (pushed) → action / navigation green net positive · red net negative · grey transfer (net-worth neutral)
